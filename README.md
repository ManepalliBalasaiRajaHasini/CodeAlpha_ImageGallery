# CodeAlpha_ImageGallery
# Image Gallery Web Application

## 📌 Project Overview

This project is a responsive Image Gallery Web Application developed using HTML, CSS, and JavaScript. It allows users to view images one by one, navigate using Previous and Next buttons, and enjoy an automatic slideshow experience.

## 🚀 Features

* Responsive Image Gallery
* Previous and Next Navigation Buttons
* Automatic Slideshow
* Image Counter Display
* Glassmorphism User Interface
* Hover Effects and Smooth Transitions
* Mobile-Friendly Design

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 🎯 How to Run

1. Download or clone the repository.
2. Place your images inside the `images` folder.
3. Open `index.html` in any modern web browser.
4. Use the Previous and Next buttons to navigate through images.

## 📸 Project Features

* Displays one image at a time.
* Automatically changes images every 3 seconds.
* Shows current image position using a counter.
* Modern Glassmorphism design for a professional appearance.
